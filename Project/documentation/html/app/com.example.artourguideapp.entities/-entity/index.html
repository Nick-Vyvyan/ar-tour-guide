Entity

abstract class Entity(name: String, url: String, latLng: LatLng, entityData: EntityData, searchId: Int)

An abstract class which is used as the template for BuildingEntity and LandmarkEntity objects, so that they can be stored and displayed together. Contains Entity data, a location, a custom dialog fragment, and an AR renderable node.
