const val ACCURACY_MAX_THRESHOLD: Float = 17.5f

Maximum accuracy threshold for anchor placement. Don't place anchors if accuracy is greater than this value. from Geospatial API documentation: "We define horizontal accuracy as the radius of the 68th percentile confidence level around the estimated horizontal location. there is a 68% probability that the true location is inside the circle. Larger numbers indicate lower accuracy."
